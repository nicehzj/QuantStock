# QuantStock 项目运行总结

---

## 📅 更新日期：2026-03-06 (高性能向量化优化与全维度网格搜索)

### 🚀 今日重大突破
1. **因子计算引擎 (Factor Engine) 性能飞跃**：
   - 引入 **“单次全量扫描 (Single-Pass Computation)”** 技术，一次 SQL 扫描即可产出多个回溯窗口（5D, 10D, 20D, 40D）的指标变体。
   - 实现了 **DuckDB 内存物化骨架**，彻底解决了 2500 万行大表的重复计算瓶颈，计算提速近百倍。
2. **高性能优化器 (Vectorized Optimizer) 落地**：
   - 利用 VectorBT 的 **Broadcasting (广播机制)** 实现了无需循环的并行参数扫描。
   - 引入了 **分批执行 (Batching)** 和显式垃圾回收，成功解决了在大规模参数空间下的 `MemoryError` 问题。
3. **防过拟合工程体系 (Anti-Overfitting)**：
   - **绝对隔离盲测**：将 2006-2023 设为优化期，2024-2026 设为绝对禁止“偷看”的最终试金石考场。
   - **参数平滑性审计**：引入了“参数高原”识别逻辑，通过邻域均值平滑锁定最优配置，有效规避了“参数孤岛”导致的伪 Alpha。
4. **横截面处理流水线**：
   - 标准化集成：MAD 去极值、实时 Rank 标准化。
   - 价格变体：支持 Close、VWAP 和 Open 三种基准的灵活切换。

### 📊 阶段性验证结果 (`smart_money_proxy`)
- **锁定配置**: Window=10D, Price=Close, Freq=5D, Direction=Reverse。
- **2024-2026 盲测绩效**: 
  - 总收益: **+48.26%** (跑赢基准)。
  - 夏普比率: **1.197**。
  - 最大回撤: **14.41%**。
  - 胜率: **70.09%**。

## 📅 更新日期：2026-03-05 (机器学习集成与止损优化)

### 🚀 重大突破
1. **机器学习集成模型 (ML Ensemble) 的实战化**：
   - 引入了 **Purging Gap (净化隔离带)**，彻底杜绝了模型在训练时对测试集的“偷看”。
   - 实施了分段训练（2006-2015, 2016-2026），适应 A 股不同历史阶段。
2. **交易控制与降噪逻辑**：
   - 引入 **Buffer-Rank Rebalancing (缓冲带调仓)**：设定 Buy-20 / Hold-40 门槛，极大降低了无效换手和手续费损耗。
   - 引入 **Individual Stop-Loss (单股止损)**：15% 强制止损机制，为复利增长提供了安全垫。

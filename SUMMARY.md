# QuantStock 项目运行总结

## 📅 更新日期：2026-03-04

### 🚀 今日重大突破
1. **工业级因子库建立**：
   - 实现了 20 个复杂量价因子（涵盖 WorldQuant 101、华泰/海通研报核心指标）。
   - 因子引擎支持参数化时间窗口，成功打通 2006-2026 的长周期计算链路。
2. **多因子对比框架升级**：
   - `vectorbt_backtest.py` 现在支持 Alpha 和 Beta 的统计剥离，能识别因子是“赚市场的钱”还是“赚选股的钱”。
   - 引入了“自动反转检测”机制，系统能自动识别 A 股的反转特性（Low-value anomaly）。
3. **机器学习集成 Alpha**：
   - 新增 `ml_ensemble_backtest.py`，利用随机森林对因子进行非线性合成。
   - 采用“长周期训练 (06-20) + 样本外测试 (21-26)”的严谨架构。
   - 通过**周频调仓**优化，将策略收益率提升至 80%+，夏普比率显著改善。

### 📊 核心绩效发现
- **A 股反转效应显著**：测试显示，绝大多数复杂因子在 A 股均表现为“反转更好”，即买入因子值最小的股票。
- **冠军因子**：`smart_money_proxy` (聪明钱代理) 在各维度测试中表现最为稳健，具有正向 Alpha。
- **机器学习威力**：ML 模型自动识别出 `amihud_illiq` 的高权重，并成功合成了具备跨周期盈利能力的策略。

### 🛠️ 技术优化
- **SQL 矩阵计算**：因子引擎全面切换至 DuckDB 窗口函数，20 个因子的全量计算控制在 15s 内。
- **命令行交互**：支持通过命令行指定任意因子进行针对性回测。
